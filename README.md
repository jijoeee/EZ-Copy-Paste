🚀 EZ Copy Paste (v1.0)
EZ Copy Paste is a professional-grade, lightweight multi-slot clipboard manager for Windows. Designed to stay out of your way as a "Ghost Dock," it allows you to manage up to 10 unique snippets effortlessly, making it the ultimate tool for data entry, coding, and customer support.

✨ Key Features
Focus Ghosting (Zero-Interruption): Uses the Windows WS_EX_NOACTIVATE API to ensure that clicking the dock never steals keyboard focus. Your cursor stays in your active document while you paste.

Edge-Snapping & Auto-Hide: Drag the dock to the top of your monitor to "snap" it. It hides automatically when not in use and slides down instantly when you hover your mouse near the top edge.

Manual Slot Entry: 🖱️ Right-click any slot to manually type or edit text. Perfect for pre-preparing scripts, links, or repetitive code blocks.

Privacy Mode: Mask sensitive snippets (passwords or API keys) with ******** using the 👁 icon. Data remains active but hidden from screen-sharing or bystanders.

Dynamic Dashboard UI: High-contrast numbering and stacked action buttons (X for clear, 👁 for privacy) provide a clean, dashboard-like experience.

Toggleable Hotkeys: Enable Ctrl + 1-0 to copy snippets into your clipboard instantly. Switch them OFF when you need those keys for normal application use.

Profile Management: Save your custom setups to .json files in the savefile folder and load them instantly for different tasks.

🛠 Quick Start Guide
Launch: Choose between 1–10 slots and click Launch New.

Auto-Copy: Highlight text in any application and click a blue Copy (X) button.

Manual-Fill: Right-click a blue button to manually type or edit text for that specific slot.

Instant Paste: Click a green button to instantly paste that slot's content into your last active application.

Hide/Show: Click the 👁 (Eye) to mask text; click the 🔒 (Lock) to reveal it.

Move: Use the ⋮⋮ handles on the far left or right to reposition the dock. Drag to the top of the screen to enable Auto-Hide.
